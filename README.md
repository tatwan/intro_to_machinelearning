# Intro to Machine Learning (مقدمة في التعلم الآلي)

Welcome to the 4-Day Machine Learning Bootcamp! This course is designed to take you from a beginner to understanding the core concepts of Machine Learning and Artificial Intelligence.

**Format:** 4 sessions × 3 hours = 12 hours total  
**Tools:** Google Colab, scikit-learn, pandas, seaborn, matplotlib

---

## Course Outline & Labs

### Day 1: Introduction to Machine Learning (مقدمة في التعلم الآلي)
What is Machine Learning? We will cover the differences between traditional programming and ML, and explore the main types of ML: Supervised Learning (التعلم الخاضع للإشراف), Unsupervised Learning (التعلم غير الخاضع للإشراف), and Reinforcement Learning (التعلم المعزز).
