# Day 2: Data is Everything

Welcome to Day 2! Today we focus on a critical truth in Machine Learning (التعلم الآلي): your model is only as good as your data.

## What we are covering today:
* **Exploratory Data Analysis (تحليل البيانات الاستكشافي):** Understanding data distributions, missing values, and identifying patterns.
* **Data Preprocessing (المعالجة المسبقة للبيانات):** Handling missing data, encoding categories, and feature scaling.
* **Feature Engineering (هندسة الميزات):** Creating new meaningful features from existing data.
* **Train/Test Splits:** Why we must separate our data into training and testing sets to evaluate models accurately.
