# Day 1: Introduction to Machine Learning (مقدمة في التعلم الآلي)

Welcome to Day 1 of the Bootcamp! Today is all about building an intuition for Artificial Intelligence (الذكاء الاصطناعي) and Machine Learning (التعلم الآلي) before diving deep into the code.

## What we are covering today:
* **What is Machine Learning?** Understanding the differences between rule-based programming and machine learning.
* **Types of Machine Learning:** 
  * Supervised Learning (التعلم الخاضع للإشراف)
  * Unsupervised Learning (التعلم غير الخاضع للإشراف)
  * Reinforcement Learning (التعلم المعزز)

## Labs and Notebooks
All our labs run directly in your browser using Google Colab. No local setup is required!

1. **[Python Crash Course](https://colab.research.google.com/github/tatwan/intro_to_machinelearning/blob/main/Day_1/notebooks/00_python_crash_course.ipynb)**
   * **What it represents:** A quick primer on Python basics.
   * **Key Concepts:** Variables, lists, pandas, and basic plotting with matplotlib.

2. **[Introduction to ML](https://colab.research.google.com/github/tatwan/intro_to_machinelearning/blob/main/Day_1/notebooks/01_intro_to_ml.ipynb)**
   * **What it represents:** Your first hands-on experience with scikit-learn.
   * **Key Concepts:** Training a classifier, learning types demo, and exploring datasets.

3. **[Linear Regression Lab](https://colab.research.google.com/github/tatwan/intro_to_machinelearning/blob/main/Day_1/notebooks/02_linear_regression_lab.ipynb)**
   * **What it represents:** Understanding how models actually "learn".
   * **Key Concepts:** Manual data fitting vs. using scikit-learn for Linear Regression (الانحدار الخطي).
