# Day 3: Core ML Algorithms

Welcome to Day 3! Today we take the intuition we've built and apply it to the most common algorithms used in the real world. We will cover both Supervised and Unsupervised Learning.

## What we are covering today:
* **Regression vs. Classification (التصنيف):** Understanding when to predict a number vs. when to predict a category.
* **Supervised Learning (التعلم الخاضع للإشراف) Models:** 
  * Linear Regression (الانحدار الخطي)
  * Logistic Regression
  * Decision Trees (أشجار القرار)
  * K-Nearest Neighbors (KNN)
* **Model Evaluation:** It's not just about accuracy! We will learn about Precision, Recall, and Confusion Matrices.
* **Clustering (التجميع):** Using K-Means to find hidden groups in data using Unsupervised Learning (التعلم غير الخاضع للإشراف).
